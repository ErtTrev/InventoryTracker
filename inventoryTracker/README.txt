Title: Inventory Tracker
Purpose: The purpose of this application is to track inventory parts as well as products that can be comprised of the same inventory parts.
Author: Eric Trevorrow
Contact Information: EricTrevorrow@gmail.com
Date: 12/2/21
IDE: IntelliJ IDEA Edu 2021.1.3
JDK: Java 11 (version 11.0.11)
JavaFX Version: Built with JavaFX-SDK-17.0.2.

Directions: To run the program, load it up within IntelliJ and click the green arrow button. You can add parts using the Add 
button underneath the Parts table and select whether the part is an internally created part or an outsourced part. 
You can modify these attributes with the Modify button under the Parts table.

You can add products by selecting the Add button underneath the Products table and can select which parts comprise the product. You can
modify these same attributes using the Modify button underneath the Products table. You can use the search bars to search for Part IDs, Product IDs,
or Name.
